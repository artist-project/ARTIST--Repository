 Reusable ARTIST Repository MDE artefacts
 Plugins: 
 - eu.artist.reusable.artefacts: 
 	Collection of reusable MDE Artefacts produced by ARTIST to support the migration of ARTIST use cases to Cloud 
 Description: 
 	Collection of reusable MDE Artefacts produced by ARTIST to support the migration of ARTIST use cases to Cloud
 	This release includes M2MT modules and libraries (ATL), UML2 Profiles and other MDE artefacts produced to support 
 	the migration of frameworks such as J2EE, .NET, Sharepoint, RCP and others
 Usage: This plugin is intended to be used as dependency by other ARTIST tool plugins
